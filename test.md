## Hotkeys

![image](img/screenshot.png)

Hotkeys can be changed in **Options**

| *Funcion*        | _Default Hotkey_            |
|------------------|-----------------------------|
| Region           | *Printscreen*               |
| Window           | Alt + PrintScreen           |
| Screen           | Ctrl + PrintScreen          |
| All screens      | No preset                   |
| Open last folder | No preset                   |
